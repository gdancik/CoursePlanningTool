# Course Planning Tool  [![flask](https://github.com/gdancik/CoursePlanningTool/actions/workflows/flask.yml/badge.svg)](https://github.com/gdancik/CoursePlanningTool/actions/workflows/flask.yml) [![nb](https://github.com/gdancik/CoursePlanningTool/actions/workflows/notebooks.yml/badge.svg)](https://github.com/gdancik/CoursePlanningTool/actions/workflows/notebooks.yml)

This is the repository for a forthcoming Course Planning Tool developed by students from [Eastern Connecticut State University](https://www.easternct.edu).

Currently, sample code is provided in the *nb* folder demonstrating how to read and modify a Word Document using the *python-docx* module.

A *flask* directory also includes example code for a flask application, which can be run using `flask run` from inside the *flask* directory.`

### Usage

Install the required packages:
```
pip install -r requirements.txt
```

### Select Files
- nb: 
	- updateSyllabus.ipynb: Jupyter Notebook with sample code
	- CSC101_Syllabus.docx: sample syllabus
- flask:
	- app.py: the flask application
	- templates: the html templates
